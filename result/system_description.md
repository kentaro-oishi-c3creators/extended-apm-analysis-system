# 拡張APM（Action Priority Matrix）分析システム 仕様解説書

本システムは、戦略コンサルタントおよび行動科学者の視点から、副業やプロジェクトの優先順位を科学的かつ直感的に決定することを目的とした分析ツールです。特に「クイックウィン（即効性のある成果）」と「長期高インパクト案件」を明確化し、精神的負荷を考慮した持続可能な行動計画を支援します。

## 1. システム概要
ユーザーが複数のアイデアを入力・評価し、独自の分析アルゴリズムによって最適な行動優先順位を導き出します。

## 2. 評価軸（10点満点）
以下の5つの軸で各プロジェクトを評価します。
- **インパクト**: 人生、収益、影響力への波及度
- **努力**: 時間、学習コスト、実行負荷
- **精神摩耗**: ストレス、対人疲労、認知負荷
- **ワクワク度**: 内発的動機、Childlike（子供のような好奇心）指数
- **凸性**: 小リスク大リターンの可能性

## 3. 分析ロジック（指標）
システムは入力された数値に基づき、以下の3つの独自指標を算出します。

| 指標名 | 計算式 | 説明 |
| :--- | :--- | :--- |
| **クイックウィン指数 (QWI)** | `(インパクト × ワクワク度 × 凸性) ÷ (努力 + 精神摩耗)` | 短期的に成果が出やすく、かつモチベーションを維持しやすい案件の指標。 |
| **長期投資指数 (LII)** | `(インパクト × 凸性) ÷ 努力` | 投資対効果が高く、将来の大きな資産になり得る案件の指標。 |
| **精神安全指数 (MSI)** | `ワクワク度 ÷ 精神摩耗` | 精神的な持続可能性を示す指標。1.0を下回る場合はバーンアウトのリスクあり。 |

### 総合スコア（ランキング用）
ランキングの決定には、以下の重み付けを用いた総合スコアを使用します。
- `総合スコア = (QWI × 0.5) + (LII × 0.3) + (MSI × 0.2)`

## 4. 出力形式と機能
- **テーブル表示**: 各プロジェクトの全スコアと詳細をリスト表示。
- **自動スコア計算**: 評価軸の数値を変更するとリアルタイムで各指数が更新されます。
- **4象限マトリクス可視化**: 「インパクト × 努力」を軸とした可視化。
    - **Quick Wins**: 高インパクト・低努力
    - **Strategic**: 高インパクト・高努力
    - **Fill-ins**: 低インパクト・低努力
    - **Thankless Tasks**: 低インパクト・高努力
- **推奨アクション**:
    - **まず着手すべき1つ**: 総合スコアが最も高い案件。
    - **今月のクイックウィン**: QWIが最も高い案件。
    - **育てるべき長期案件**: LIIが最も高い案件。
    - **捨てる候補**: インパクトが低く、努力が高い案件。

## 5. AI連携機能（Gemini搭載）
- **最小着手ステップの提案**: 30分以内で完了できる具体的アクションをAIが提案。
- **全体行動指針の生成**: ユーザーのパーソナリティとプロジェクト状況に合わせた戦略的アドバイスを提供。
- **AI戦略レポート出力**: 分析結果に基づいたフルレポートをMarkdown形式でダウンロード可能。

## 6. その他の特徴
- **データ管理**: JSONおよびCSV形式でのエクスポート・インポートに対応。
- **フィロソフィー**: 常に「Keep Touching Childlike?（ワクワクを忘れていないか？）」という問いをユーザーに提示し、内発的動機を重視します。
- **編集可能な構造**: 結果を再編集可能な形式で管理し、プロジェクトの進捗（0-100%）も追跡可能です。
