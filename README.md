# 拡張APM（Action Priority Matrix）分析システム

行動科学と戦略コンサルティングの知見を融合させた、副業や個人プロジェクトのための意思決定支援ツールです。
単なる「重要度×緊急度」の分析を超え、ワクワク度や精神的負荷、リターンの凸性を考慮した「科学的かつ直感的な」優先順位付けを実現します。

## 🚀 主な特徴

- **多角的な評価軸**: インパクト、努力、精神摩耗、ワクワク度、凸性の5つの軸でアイデアを分析。
- **3つの独自指標**:
    - **QWI (Quick Win Index)**: 今すぐ着手すべき「即効性」の高い案件。
    - **LII (Long-term Investment Index)**: 将来の大きな資産となる「長期投資」案件。
    - **MSI (Mental Safety Index)**: モチベーション維持のための「精神安全」指標。
- **データ可視化**:
    - **4象限マトリクス**: インパクトと努力のバランスを俯瞰。
    - **属性バランス分析（レーダーチャート）**: 案件の強み・弱みを一目で把握。
- **AI外部連携プロンプト生成**: ChatGPTやGeminiの**有料サブスクリプション環境**に最適化された専用プロンプトを生成。APIキー不要で最新・最強のAI分析を受けられます。
- **ローカル完結・オートセーブ**: 入力データはブラウザのローカルストレージに自動保存されます。外部サーバーにデータが送られることはありません。

## 使い方

1.  **新規アイデア追加**: 右上の「新規アイデア追加」ボタンからプロジェクトを入力します。
2.  **パラメータ調整**: 5つのスライダーを動かして、各数値を評価します。
3.  **分析結果の確認**: リストの順位やマトリクスを見て、どのプロジェクトに注力すべきかを確認します。
4.  **AIに相談する**:
    -   設定ボタン（歯車アイコン）からご自身のパーソナリティを選択。
    -   「AI連携プロンプトを生成」をクリック。
    -   使い慣れたChatGPTやGeminiの画面に貼り付けて、具体的なアドバイスを得ます。

## 開発者向け（ローカル実行方法）

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

起動後、 `http://localhost:3000` にアクセスしてください。

## 技術スタック

- **Frontend**: React 19, Vite, TypeScript
- **Styling**: Vanilla CSS (TailwindCSS v4)
- **Visualization**: Recharts
- **Components/Icons**: Lucide React, Framer Motion
